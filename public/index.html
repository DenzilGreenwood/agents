📚 About This Demo

This interactive demonstration showcases CIAF's agentic workflow capabilities:

  • Identity & Access Management (IAM): Role-based permissions with ABAC conditions
  • Multi-Tenant Isolation: Agents can only access resources from their own tenant
  • Privilege Elevation (PAM): Time-bound grants for sensitive operations
  • Evidence Vault: Cryptographically signed audit trail
  • Compliance: HIPAA, GDPR, SOX obligation tracking

💻 Quick Start Code

# Install CIAF
pip install pyciaf==1.3.1

# Initialize components
from ciaf.agents import IAMStore, PAMStore, PolicyEngine, EvidenceVault, ToolExecutor

iam = IAMStore()
pam = PAMStore()
vault = EvidenceVault(signing_secret="your-secret")
policy = PolicyEngine(iam, pam, sensitive_actions={"delete_data"})
executor = ToolExecutor(policy, vault, pam)
# Create action request
request = ActionRequest(
    action="read_data",
    resource=dataset,
    params={"dataset_id": "dataset-2026-q1"},
    justification="Q1 analysis",
    requested_by=agent
)

# Execute with authorization
result = executor.execute_tool("read_data", request)

if result.allowed and result.executed:
    print(f"✓ Success: {result.result}")
else:
    print(f"✗ Denied: {result.reason}")
# Grant privilege elevation for 30 minutes
grant = pam.create_grant(
    principal_id="agent-001",
    elevated_role="data_admin",
    duration_minutes=30,
    approved_by="manager",
    ticket_reference="TICKET-2026-001",
    purpose="Emergency data deletion"
)

# Verify chain integrity
chain_valid = vault.verify_chain()
print(f"Evidence chain valid: {chain_valid}")
